# Boston Housing Price Prediction

Predicting house prices in Boston using Linear Regression.

---

## Dataset
**Dataset:** Boston Housing Dataset  
**Features analyzed:** RM (average number of rooms), LSTAT (lower status of the population), TAX (property tax rate), NOX (air pollution level)

---

## Goal / Learning Outcome
This project helped me practice:

- Building and training a Linear Regression model
- Analyzing features that influence house prices
- Understanding basic machine learning workflow

---

## How to Run
Open the notebook (`boston_housing_model.ipynb`) in **Jupyter Notebook** or **Google Colab**.
